# karts
JAVA: Arrays de objetos

**Simulación de una carrera de Karts**.

## Introducción

Para ejecutar en Netbeans, crear un proyecto nuevo y copiar dentro la carpeta `src`.

## Diagrama de clases

![Diagrama de clases](img/kartsUML.png)

Los valores que se establecen para los karts son:
- Valor por defecto para el depósito: 30 decilitros.
- Rango de velocidad: de 40 a 50.
